# danaottaviani.com

A personal website and blog.

## Built With

 - [Gridsome](https://gridsome.org/)

## Installation

1. Clone the repo: `git clone https://github.com/Dana94/website.git`
1. `cd website`
1. `npm install`
1. `gridsome develop` to start a local dev server at `http://localhost:8080`
